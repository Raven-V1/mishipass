Public-facing Beta overview

MishiPass

Privacy-first dynamic QR passport and recovery system for cats.

One permanent QR. Different mode when the cat needs it.

/c/MP-XX-XXXX-XXXX

How It Works

  1. 01 Register cat

    Create an owner-controlled record in the configured Worker backend.

  2. 02 Generate permanent ID

    Use one public MishiPass ID and QR route for the cat.

  3. 03 Switch active mode

    The same QR route resolves to the current mode when scanned.

Beta Modes

Active Profile

A simple public profile with mode-appropriate, owner-approved contact visibility.

Missing Alert

A recovery-focused public view with last-seen context and optional reward visibility.

Vet Visit

A controlled visit workflow for temporary records, while medical/cartilla data stays private.

Privacy Model

Owner-controlled privacy is built into the public route model.

  • No internal database IDs on public pages.
  • Owner controls contact visibility.
  • Medical/cartilla data remains private.
  • Medication entries remain documentation-only records.

Runtime Boundary

The public site is hosted on GitHub Pages. The QR/API runtime is handled by a TypeScript Cloudflare Worker with D1.
